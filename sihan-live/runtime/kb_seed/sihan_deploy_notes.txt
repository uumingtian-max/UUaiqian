# 赵思涵 Live（阿权私有服务器）— 部署与常用命令

## 对外端口（NAT）
- 61930 → 本机 8000：API 网关（聊天、TTS 代理、知识库导入、/deploy/run）
- 61931 → 本机 8001：前端 Nginx 静态页

## 目录
- 项目根目录（默认 shell cwd）：/home/linux/sihan-final
- 网关与伴读：/home/linux/sihan-final/runtime/
- 知识库导入放这里：/home/linux/sihan-final/runtime/companion_data/kb_import/
- 记忆 JSON：/home/linux/sihan-final/runtime/companion_data/memory_nac.json

- sudo systemctl restart sihan-gateway.service
- sudo systemctl restart sihan-tts.service

cd /home/linux/sihan-final && docker-compose build frontend && docker rm -f sihan-final_frontend_1 2>/dev/null; docker-compose up -d frontend

POST /kb/ingest-directory，JSON：{"path":"/home/linux/sihan-final/runtime/companion_data/kb_import"}

## 小涵通过聊天执行服务器命令
模型在回复里写 ```deploy ... ``` bash 块，网关会执行并把输出附在回复末尾。
或直接 POST /deploy/run，body：{"command":"...", "cwd":"/home/linux/sihan-final", "timeout_sec":300}

## 直链抓取（未收录页 / 可选内网）
当用户消息含 `http://` 或 `https://` 时，网关会用本机发 GET 抓取正文片段（HTML 提纯文本），写入伴读「直链页面摘录」。
环境变量（写在 sihan-gateway.service 的 Environment=）：
- `SIHAN_FETCH=1` 开启（默认 1）；`SIHAN_FETCH=0` 关闭。
- `SIHAN_FETCH_TIMEOUT=14` 单页超时秒。
- `SIHAN_FETCH_MAX_BYTES=2000000` 单页最大字节。
- `SIHAN_FETCH_MAX_URLS=5` 每轮最多跟几个链接。
- `SIHAN_FETCH_ALLOW_PRIVATE=1` 允许访问 RFC1918 内网、localhost（**仅私有服务器且你自担风险时**打开）。
- `SIHAN_FETCH_COOKIE=...` 浏览器里复制的 Cookie 字符串，用于需登录才有内容的页面（遵守站点条款）。
- `SIHAN_FETCH_UA=...` 可自定义 User-Agent。
